# Инструкции для `docs/reference/true_api/original`

- Храните только официальные файлы True API в формате DOCX/PDF/ZIP.
- Не переименовывайте файлы внутри: имя должно отражать версию и дату.
- При замене файла сохраняйте предыдущую версию под другим именем, если она используется.
- Перед коммитом проверяйте, что бинарные файлы не повреждены (открываются локально).
