Заказ кодов маркировки

Фронтенд-прототип демонстрирует раздельные модули: загрузку карточек из Национального каталога, выбор позиций и оформление заказа КМ. Данные подтягиваются по периоду обновления карточек (по умолчанию — 3 года).

1. Запрос карточек

Укажите период обновления карточек Национального каталога и, при необходимости, фильтр по названию или GTIN.

Если не задавать период, автоматически будет запрошен диапазон обновлений за последние три года.

2. Карточки из Национального каталога

Список обновляется после запроса. Данные группируются по последнему статусу карточки.
